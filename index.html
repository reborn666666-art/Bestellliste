⚙️ Einmalige Einrichtung

Für die Team-Synchronisierung wird ein kostenloser Online-Speicher (JSONBin.io) benötigt. Diese Einrichtung muss nur einmal von einer Person gemacht werden – danach einfach die fertige Datei ans Team weitergeben.

Schritt 1 – Account anlegen
Gehe auf jsonbin.io → oben rechts „Sign Up" → E-Mail & Passwort eingeben → registrieren → E-Mail bestätigen.
Schritt 2 – API Key erstellen
Nach dem Login: oben rechts auf dein Profilbild klicken → „API Keys" → „+ CREATE API KEY" → beliebigen Namen eingeben (z.B. Bestellliste) → „Create" → den erscheinenden Key (beginnt mit $2a$10$) vollständig kopieren.
Schritt 3 – Bin (Datenspeicher) erstellen
Im linken Menü auf „Bins" klicken → „+ CREATE BIN" → im Textfeld alles löschen und genau diesen Text einfügen:

{"orders":[]}

→ „Create Bin" klicken → die Bin ID erscheint oben in der URL oder im Bin-Header (lange Zeichenkette, z.B. 6849abc123def456) → kopieren.
Schritt 4 – Hier eintragen und speichern
API Key und Bin ID unten eintragen → „Speichern & Starten" klicken. Die Datei ist dann für das ganze Team einsatzbereit.
💡 Damit niemand etwas eingeben muss: Öffne die HTML-Datei in einem Texteditor (Rechtsklick → „Öffnen mit" → Notepad). Suche nach PRESET_API_KEY und PRESET_BIN_ID und trage dort deine Werte ein. Dann speichern und die Datei ans Team schicken – fertig.
